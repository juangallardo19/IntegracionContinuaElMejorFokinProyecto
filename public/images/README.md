# Imágenes de la Aplicación

Esta carpeta contiene las imágenes utilizadas en la aplicación.

## Archivos Requeridos

Por favor, coloca la siguiente imagen en esta carpeta:

1. **technology-background.jpg** - Imagen de fondo para la sección de Tecnología e Informática

## Especificaciones

### technology-background.jpg
- Dimensiones: 1152 × 768 píxeles
- Formato: JPG o PNG
- Tema: Tecnología e informática (computadoras, código, circuitos, etc.)
- Estilo: Imagen educativa y amigable para niños
- Tonos: Preferiblemente colores suaves que no dificulten la lectura del texto

Si la imagen no está presente, la vista mostrará un fondo gris por defecto.
