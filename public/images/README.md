# Imágenes de la Aplicación

Esta carpeta contiene las imágenes utilizadas en la aplicación.

## Archivos Requeridos

Por favor, coloca las siguientes imágenes en esta carpeta:

1. **technology-background.png** - Imagen de fondo para la sección de Tecnología e Informática
2. **art-background.png** - Imagen de fondo para la sección de Artística
3. **logic-background.png** - Imagen de fondo para la sección de Pensamiento Lógico

## Especificaciones

### Todas las imágenes de fondo
- Dimensiones: 1152 × 768 píxeles
- Formato: PNG con transparencia
- Fondo: Transparente (el fondo de color existente se mantendrá)
- Contenido: Solo ilustraciones y elementos gráficos sin fondo

### technology-background.png
- Tema: Tecnología e informática (computadoras, código, circuitos, etc.)
- Estilo: Ilustraciones educativas y amigables para niños

### art-background.png
- Tema: Arte y creatividad (pinceles, colores, paletas, etc.)
- Estilo: Ilustraciones coloridas y creativas

### logic-background.png
- Tema: Pensamiento lógico (patrones, rompecabezas, formas geométricas)
- Estilo: Ilustraciones educativas con elementos geométricos

## Notas

- Usa PNG con transparencia para que se vean solo las ilustraciones
- Si las imágenes no están presentes, las vistas mostrarán el fondo de color por defecto
- Los tonos suaves en las ilustraciones ayudan a mantener la legibilidad del texto
